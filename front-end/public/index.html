Que dados podem ser monitorados?

Memória

A memória é um componente extremamente importante em um servidor, sem ela é impossivel armazenar dados bem como impacta diretamente na execução de determinados aplicativos. Além disso, ela é a base para que a CPU execute bem sua função.

Memória

CPU

Trabalha juntamente com a memória, a CPU pode ser considerada o cérebro do servidor, é responsável por buscar e processar instruções que estão armazenadas na memória, controlar o fluxo de informações e gerenciar recursos.

CPU

Disco

O disco tem a função de armazenar e recuperar dados, os dados guardados nesse componente ficam guardados de maneira permanente, mesmo quando o servidor está desligado. Quando falamos de escalabilidade essa peça é parte principal do projeto.

Disco

Rede

O mundo hoje é conectado pela internet, de nada adianta ter um computador com as melhores peças se conexão é instavel e não permite que as informações sejam transportadas na velocidade adequada e o servidor acaba sendo sub-utilizado.

Rede
